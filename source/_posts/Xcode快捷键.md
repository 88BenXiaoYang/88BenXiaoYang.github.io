```
缩写说明:
cmd:command;  ctr:control;  sft:shift

cmd + r: 运行
cmd + .: Stop Run
cmd + [: 左缩进
cmd + ]: 右缩进
cmd + /: 注释或取消注释
cmd + \: 设置或取消断点
Esc : 显示代码提示菜单
cmd ＋ ←: 移动光标到本行行首  
cmd ＋ →: 移动光标到本行行尾
sft ＋ cmd ＋ ←: 选择到行首
sft ＋ cmd ＋ →: 选择到行尾
ctr + K: 删除本行
ctr + cmd + 上下箭头: 切换.h和.m文件
ctr + cmd + ←: 回到上一级界面
ctr + cmd + →: 回到下一级界面
cmd + sft + F: 项目中查找
sft + 左右箭头: 向左或向右选择一个字符
ctr + sft + 左右箭头: 向左或向右选择一个词汇
```
